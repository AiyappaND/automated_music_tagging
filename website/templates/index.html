Base API url: /api/v1.0

Show Featured Songs

Returns Json data about 40 featured songs on Spotify along with the classified activity

  • URL

    • /songs

  • Method

    • GET

Category Songs

Returns Json data about 10 songs of the category

  • URL

    • /cat/category_name

  • Method

    • GET

  • Parameters

    • category_name can be any one of [party, sleep, workout, dinner]

Search Songs

Returns Json data about 2 songs with the classified activity

  • URL

    • /search?track={track name}

  • Method

    • GET

  • Parameters

    • track=[string]
